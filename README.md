# Pomyo 🍅

심플한 포모도로 타이머 PWA

## 기능

- 타이머 원을 드래그해서 시간 설정 (1~60분)
- 탭해서 시작 / 일시정지
- 포커스 모드 (10초 무조치 시 자동 전환)
- 완료 시 소리 및 진동 알림
- 기록 탭에서 달력으로 세션 히스토리 확인
- 테마 색상 5가지 선택
- 시계방향 / 반시계방향 설정
- PWA 설치 지원 (홈 화면 추가)

## 설치 및 실행

정적 파일이므로 별도 빌드 없이 바로 사용 가능합니다.

```bash
npx serve .
# 또는
python3 -m http.server 8080
```

## 파일 구조

```
├── index.html     # 앱 전체
├── manifest.json  # PWA 매니페스트
├── sw.js          # 서비스 워커
├── icon-192.png   # 앱 아이콘
└── icon-512.png   # 앱 아이콘
```
